# Laboratorio de Sistemas Complejos
### Explorando el orden oculto en el caos

Sitio web de divulgación científica sobre sistemas complejos, desarrollado 
como proyecto de retribución social de la Maestría en Sistemas Complejos.

🌐 **[jardigarci.github.io/laboratorio-sistemas-complejos](https://jardigarci.github.io/laboratorio-sistemas-complejos)**

---

## ¿Qué encontrarás aquí?

Una introducción visual e interactiva a las tres grandes herramientas 
para estudiar sistemas complejos:

- **Análisis Fractal** — dimensión fractal, box-counting interactivo, 
  fractales sintéticos generados en tiempo real, y fractales en la naturaleza
- **Análisis Multifractal** — espectro f(α), cascada binomial, MF-DFA
- **Análisis Topológico de Datos** — homología persistente, filtraciones 
  Vietoris–Rips, diagramas de persistencia
- **Redes Complejas** — métricas de centralidad, modelo Watts–Strogatz 
  interactivo, redes libre de escala

El contenido está diseñado para dos niveles simultáneos: accesible para 
público general, con secciones técnicas expandibles para estudiantes de 
licenciatura y posgrado.

---

## Tecnologías

- HTML5 · CSS3 · JavaScript vanilla
- Canvas API para fractales generativos
- D3.js para el mapa conceptual interactivo
- Sin frameworks ni dependencias de servidor — funciona estático en GitHub Pages
